思科 MCP — Cisco Packet Tracer 自动配置

📖 使用说明（首次使用必看）
  1. 打开 Cisco Packet Tracer 8.2.1
  2. Extensions → Scripting → Configure PT Script Modules...
  3. 点击 Add，选择 extension/cisco-pt-mcp.pts
  4. 重启 Packet Tracer
  5. 在 PT 中 Extensions → Packet Tracer MCP 打开桥接窗口（状态应显示 connecting）
  6. 回到本页面，点击 「连接 PT」
